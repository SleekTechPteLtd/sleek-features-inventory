# Accounting domain inventory

This folder is the **Accounting** TLD in the Sleek features inventory.
Features are grouped by **application surface** matching `by-surface/`.

## Layout

| Folder | Surface |
|--------|---------|
| `admin-app/` | Sleek Website admin routes; admin-only accounting flows |
| `customer-app/` | Customer-facing bookkeeping UI (customer-main, customer-common, customer-root) |
| `sleekbooks/` | ERPNext/Frappe/SleekBooks integrations; ledger data pipelines |
| `coding-engine/` | acct-coding-engine, sleek-receipts, supplier-rules-service, document/coding pipeline |
