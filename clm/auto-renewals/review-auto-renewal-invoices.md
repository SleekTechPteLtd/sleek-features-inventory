## Evidence

### Component
`src/pages/AutoRenewals/AutoRenewalsList.jsx`

- Fetches invoices via `sleekBillingsApi.getInvoices()` with a fixed filter: `{ invoiceOrigin: "autoRenewal" }` plus a text search regex over `number` and `title` fields.
- Search is debounced (500 ms); resets to page 1 on new query.
- Pagination: 100 items per page (`ITEMS_PER_PAGE = 100`), using MUI `<Pagination>`.
- Table columns: **Title**, **Amount** (SGD), **Status** (paid = green badge, other = yellow), **Issue Date**, **Payment Date**, **Auto-Renewal Attempt** (shows `autoRenewalChargeAttemptedAt`, `autoRenewalChargeAttemptedBy`, and `autoRenewalChargeError` if present).
- "View Company" link opens `VITE_ADMIN_APP_URL/admin/company-overview/?cid={companyId}&currentPage=Billing+Beta` in a new tab.
- Manual auto-charge button (`handleManualAutoCharge`) wired to `sleekBillingsApi.triggerManualAutoCharge(subscriptionId)` but entire dropdown is commented out.

### API calls (`src/services/api.js`)
- `getInvoices(options)` → `GET /invoices/?{page,limit,filter}` — returns `{ docs, totalDocs }` (paginated invoice list).
- `triggerManualAutoCharge(companyId)` → `POST /subscription-auto-renewal/trigger-manual-auto-renewal-charge` with `{ companyId }` — manually re-triggers auto-renewal charge for a subscription.
- `getEligibleRenewals(companyId)` → `GET /subscription-auto-renewal/eligible/{companyId}` — not called from this page but part of the same auto-renewal surface.

### Route
`src/App.jsx` line 165: `<Route path="auto-renewals" element={<AutoRenewalsList />} />`

### Auth surface
Axios instance sends `Authorization: Bearer <token>` (OAuth) or raw token (alternate login), plus `App-Origin: admin` or `admin-sso`. Implies internal admin-only access.

### Backend controller
`sleek-clm-monorepo/apps/sleek-billings-backend/src/invoice/controllers/invoice.controller.ts`
- `GET /invoices/` — `@Auth()` (any authenticated user)

`sleek-clm-monorepo/apps/sleek-billings-backend/src/customer-subscription/controllers/subscription-auto-renewal.controller.ts`
- `POST /subscription-auto-renewal/trigger-manual-auto-renewal-charge` — `@Auth()` + `@GroupAuth(BillingSuperAdmin, BillingOperationsAdmin, SalesAdmin)`
- `GET /subscription-auto-renewal/eligible/:companyId` — `@Auth()`

### Invoice schema
`sleek-clm-monorepo/apps/sleek-billings-backend/src/invoice/models/invoice.schema.ts`
- `InvoiceOrigin.autoRenewal` — enum value used as the list filter
- `autoRenewalChargeAttemptedAt?: Date` — timestamp of charge attempt
- `autoRenewalChargeAttemptedBy?: string` — actor who triggered charge
- `autoRenewalChargeError?: string` — latest single error string
- `autoRenewalChargeErrors?: { paymentMethodId: string; error: any }[]` — per-payment-method error array
- `status: InvoiceStatus` — `paid`, `authorised`, `failed`, `voided`, etc.
- MongoDB collection: `invoices`
